# weather-js-project

A modern weather application built with vanilla JavaScript, HTML, and CSS. Displays current conditions, hourly and 5-day forecasts using your GPS location.

## Features

- GPS-based location detection with fallback to Richmond Hill, Ontario
- City search with autocomplete
- Current weather with temperature, humidity, wind, pressure, UV index, and visibility
- Hourly forecast (next 24 hours)
- 5-day daily forecast
- Sunrise/sunset with visual sun arc
- Toggle between Celsius and Fahrenheit
- Environment Canada hazard warnings
- Responsive design for mobile and desktop
- Glassmorphism card-based UI

## Powered by

- [WeatherAPI.com](https://www.weatherapi.com/) for weather data
- [BigDataCloud](https://www.bigdatacloud.com/) for reverse geocoding
